# Markdown Grammer

## 주요 기능
### 제목
앞에 #을 붙입니다.
```
# <h1> 태그와 같다.
## <h2> 태그와 같다.
### <h3> 태그와 같다.
```

### 코드쓰기
한 줄일 경우 `내용`, 여러 줄일 경우 ```**(해당언어)** 내용``` 또는 ~~~내용~~~
<pre> 
```javascript
    var test;
    function test() {
        return false;
    }
```
</pre>
